##### hc-slider
区间选择控件！

###### 普通用法

``` html
  <hc-slider value="32"></hc-slider>
```
<div class="phone">
  <hc-slider value="32"></hc-slider>
</div>

###### 不同尺寸
`size`控制滑块的尺寸大小，`mini`、`small`、`default`、`large`

``` html
  <hc-slider size="mini" value="32"></hc-slider>
  <hc-slider size="small" value="32"></hc-slider>
  <hc-slider size="default" value="32"></hc-slider>
  <hc-slider size="large" value="32"></hc-slider>
```
<div class="phone">
  <hc-slider size="mini" value="32"></hc-slider>
  <hc-slider size="small" value="32"></hc-slider>
  <hc-slider size="default" value="32"></hc-slider>
  <hc-slider size="large" value="32"></hc-slider>
</div>

###### 自定义主题
`color`设置滑块选中的颜色！

``` html
  <hc-slider color="#f00" value="32"></hc-slider>
```
<div class="phone">
  <hc-slider color="#f00" value="32"></hc-slider>
</div>

###### 自定义区间
`min`设置最小值，`max`设置最大值！

``` html
  <hc-slider min="10" value="20" max="80"></hc-slider>
```
<div class="phone">
  <hc-slider min="10" value="20" max="80"></hc-slider>
</div>

###### 步长
`step`设置每次滑动的距离为`step`的倍数！

``` html
  <hc-slider step="10"></hc-slider>
```
<div class="phone">
  <hc-slider step="10"></hc-slider>
</div>

###### 只读
`readonly`设置只读模式，不可拖拽！

``` html
  <hc-slider value="32" readonly></hc-slider>
```
<div class="phone">
  <hc-slider value="32" readonly></hc-slider>
</div>

###### 禁用
`disabled`设置为禁止拖拽状态

``` html
  <hc-slider value="32" disabled></hc-slider>
```
<div class="phone">
  <hc-slider value="32" disabled></hc-slider>
